# 🎲 Pig Game

A fun dice game where players compete to reach a target score.

## 🎮 How to Play
- Roll the dice
- Hold to save score
- First player to reach target wins

## 🚀 Features
- Two-player mode
- Score tracking
- Interactive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run

1. Clone the repository  
2. Open the project folder  
3. Open "index.html" in your browser  

OR use VS Code Live Server
