# Pig Game

A fun dice game where players compete to reach a target score.

## 🎮 How to Play
- Roll the dice
- Hold to save score
- First player to reach target wins

## 🚀 Features
- Two-player mode
- Score tracking
- Interactive UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run

1. Open the project folder
2. Double click "index.html"

OR (recommended):
Use VS Code Live Server
